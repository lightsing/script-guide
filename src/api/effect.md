# 效果


## EffectType

目前游戏有以下效果类型，你需要通过全局常量来访问它们：

### EffectType.Taunt

嘲讽

### EffectType.Frozen

冻结

### EffectType.Fatigue

休整

### EffectType.Shield

圣盾

### EffectType.Poisoned

中毒

### EffectType.Steal

潜行

### EffectType.Windfury

风怒

## EffectAttrs

效果属性，目前只包含 `duration`，即持续事件。
